# PM++ Documentation

PM++ is a JAX-based differentiable Particle-Mesh cosmology package for serial and multi-GPU N-body simulations.

```{warning}
PM++ is research software. Interfaces, correction models, and multi-GPU runtime details may change until a stable release is declared.
```
